=========
 Welcome
=========

These pages provide instructions and advice for the practical tutorial session
on the NLP Interchange Format (NIF) at the Second Summer Datathon on Linguistic
Linked Open Data (SD-LLOD-17).

The NLP Interchange Format (NIF) is an RDF/OWL-based format that aims to achieve
interoperability between Natural Language Processing (NLP) tools, language
resources and annotations.
